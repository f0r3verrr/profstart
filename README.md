# ПРОФСТАРТ+

Одностраничный информационный сайт ООО «Профстарт+» — профориентационные маршруты по химическим предприятиям региона.

## Содержание

- О компании (цель, миссия)
- Партнёры
- Маршруты
- Контакты
- Реквизиты

## Технологии

- HTML5, CSS3 (адаптивная вёрстка), JavaScript (без фреймворков)
- Шрифты: Montserrat, Unbounded (Google Fonts)

## Локальный просмотр

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
# Python 3
python -m http.server 8000

# или npx
npx serve .
```

Сайт будет доступен по адресу http://localhost:8000

## GitHub Pages

Репозиторий: [github.com/f0r3verrr/profstart](https://github.com/f0r3verrr/profstart)

**Включить сайт как GitHub Pages:**

1. Откройте репозиторий на GitHub → **Settings** → **Pages**.
2. В блоке **Build and deployment** в поле **Source** выберите **Deploy from a branch**.
3. В **Branch** выберите ветку `main` и папку **/ (root)**.
4. Нажмите **Save**.

Через 1–2 минуты сайт будет доступен по адресу: **https://f0r3verrr.github.io/profstart/**
